## OpenCover to LCOV

The XSLT file [opencover-to-lcov.xslt](./opencover-to-lcov.xslt) converts OpenCover report to LCOV using XSLT 3.0 and is licensed under the MIT License.  
This repository was created as a response to [OpenCover Issue #643](https://github.com/OpenCover/opencover/issues/643).

This is derivative work done based on [ReportGenerator](https://github.com/danielpalme/ReportGenerator) by [Daniel Palme](https://github.com/danielpalme).
